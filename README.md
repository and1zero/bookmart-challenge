# Tusitala Challenge Stage 2

Challenge for Tusitala's BookMart

## Running Locally
```
bash server
```

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```

Live demo can be observed [here](http://warm-springs-5019.herokuapp.com/).
